HTML/CSS

El trabajo final consiste en crear un portafolio en HTML+CSS el cual contenga una descripción general de los trabajos desarrollados en el curso de herramientas.

}
  1. El portafolio es individual y deberá estar publicado en la página personal de GitHub y se deberá acceder desde el directorio de páginas ttps://herramientasdsw.github.io/.
  2. El portafolio deberá tener una página de inicio con:
    • Una presentación del alumno
    • Una descripción general del portafolio
  3. El portafolio deberá tener una página adicional para cada herramienta vista en clase:
    • Scratch
    • Lego
    • Blender
    • Unity
    • HTML/CSS
  4. Cada página deberá tener un estilo adecuado y estar compuesta por:
    • menú de encabezado
    • contenido
    • pie de página
  5. El menú deberá tener enlaces al resto de las páginas
  6. El contenido de cada página deberá tener:
    • Un titular con el nombre del trabajo
    • Un párrafo con la descripción del trabajo
    • una lista no ordenada con las características del trabajo
    • Una o varias imágenes del trabajo realizado
    • Un párrafo con la opinión personal acerca del trabajo
